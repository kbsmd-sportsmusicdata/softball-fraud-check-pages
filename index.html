        Data Dictionary
        Definitions for all metrics used in this dashboard. PEARatings proprietary metrics are labeled accordingly. Dashboard-computed fields are noted where applicable.
        Dashboard Intelligence
        Fraud Delta
        FD — Dashboard Computed
        Actual Win% minus Pythagorean Win%. Positive values indicate outperformance (running hot); negative values indicate underperformance (running cold) relative to run differential.
        Fraud Tier
        Classification — Dashboard Computed
        Categorical label based on Fraud Delta thresholds. Regression Candidates: FD > threshold (winning beyond run-differential expectation). True Contenders: FD near zero (record matches metrics). Late-Season Risers: FD below threshold (underperforming relative to underlying stats).
        SOS
        Strength of Schedule
        A measure of the average quality of opponents faced. Higher values indicate a tougher schedule. Scale: 0.0 (weakest) to ~0.8 (elite).
        Rank
        Composite Ranking Proxy
        Composite system ranking from 1 to 304. Used as a proxy for USA Softball top-25 rankings since no official list is embedded in the dataset. Teams ranked 1–25 represent the estimated contender field.
        Strength Score
        Composite Strength Metric
        Composite metric combining multiple performance dimensions (TSR, SOS, record quality) into a single strength estimate. Used to sort teams in the Volatility Grid top-40 view.
        NET
        Net Win Rate
        Decimal win-rate metric ranging from 0 to 1. Represents overall winning percentage in a normalized format, factoring in game context where applicable.
        PEARatings Core Metrics
        TSR
        Team Strength Rating
        PEAR's primary composite team quality metric. Synthesizes pitching, offense, and schedule context into a single strength estimate. Higher is better.
        WAR
        Wins Above Replacement (PEAR Approx.)
        PEAR's approximation of Wins Above Replacement. Estimates how many additional wins a team contributes compared to a replacement-level baseline in context of the dataset.
        WPOE
        Win Percentage Over Expected
        Measures "luck" — the gap between actual win percentage and what the team's underlying metrics would predict. A high WPOE signals positive variance; a low WPOE signals likely regression.
        PYTHAG
        Pythagorean Win Percentage
        Expected win percentage derived from runs scored (RF) and runs allowed (RA): RF² ÷ (RF² + RA²). Predicts what a team's record "should" be based purely on run differential.
        KSHOTS
        Killshots
        An inning in which a team scores 2 or more runs while allowing zero. Represents game-changing, high-leverage scoring bursts that often decide outcomes.
        CONCD
        Conceded Killshots
        Killshot innings surrendered to the opponent. Measures defensive/pitching vulnerability in high-leverage situations.
        KSHOT R
        Killshot Ratio
        Ratio of Killshots scored to Conceded Killshots. Values above 1.0 indicate the team creates more game-deciding bursts than it concedes. Key indicator of clutch performance.
        KSHOTS/G
        Killshots Per Game
        Average number of Killshot innings scored per game. Normalizes raw killshot totals to per-game rate to allow fair comparison across teams with different game totals.
        CONCD/G
        Conceded Killshots Per Game
        Average number of Killshot innings surrendered per game. Lower values indicate more consistent defensive performance in high-leverage moments.
        Pitching & Defense
        ERA
        Earned Run Average
        Average number of earned runs allowed per 7 innings pitched (softball standard). Lower ERA = better pitching performance. Unearned runs (via errors) are excluded.
        WHIP
        Walks + Hits Per Inning Pitched
        Measures baserunner efficiency: (Walks + Hits) ÷ Innings Pitched. A WHIP below 1.0 is elite; above 1.5 signals significant baserunner exposure. Lower is better.
        K/7
        Strikeouts Per 7 Innings
        Strikeout rate normalized to a 7-inning softball game. Higher values indicate a more dominant strikeout pitcher. Allows comparison regardless of total innings pitched.
        PCT
        Fielding Percentage
        Proportion of defensive plays converted without an error: (Putouts + Assists) ÷ (Putouts + Assists + Errors). Higher is better. The softball standard for elite defense is .970+.
        Offense
        RPG
        Runs Per Game
        Average runs scored per game. Primary input for Pythagorean Win% calculations in this dashboard. Used as proxy for Runs For (RF) when total RF is unavailable.
        BA
        Batting Average
        Hits ÷ At Bats. The classic measure of contact hitting ability. Does not capture walks or extra-base value. Often used alongside OBP and SLG for a fuller offensive picture.
        OBP
        On Base Percentage
        (Hits + Walks + HBP) ÷ (At Bats + Walks + HBP + SF). Measures how frequently a batter reaches base by any means. OBP is a stronger predictor of run scoring than BA alone.
        SLG
        Slugging Percentage
        Total Bases ÷ At Bats. Weights hits by their extra-base value (1B=1, 2B=2, 3B=3, HR=4). Measures raw power output. Higher values indicate more extra-base production.
        OPS
        On Base Plus Slugging
        OBP + SLG. Simple but powerful combined offensive metric capturing both plate discipline and power. A team OPS above .800 is considered strong in collegiate softball.
        Rankings & Record Context
        ELO
        ELO Rating
        Adaptive ranking system where teams gain/lose points based on game outcomes and opponent strength. Self-correcting over time — a strong rating reflects sustained performance against quality competition.
        RPI
        Ratings Percentage Index
        Weighted formula: 25% Win% + 50% Opponents' Win% + 25% Opponents' Opponents' Win%. Heavily influenced by schedule quality. Commonly used in NCAA selection criteria.
        RQI
        Record Quality Index
        Composite metric evaluating the quality of a team's wins and losses. A team with a .500 record against top-25 opponents will rank higher than a .500 team that played weaker opposition.
        Q1 – Q4
        Quadrant Records
        Win-loss record split by opponent quality quadrant. Q1 = top opponents (most predictive for NCAA selection); Q4 = weakest opponents. NCAA uses quadrant records to evaluate resume quality and compare teams.
